# Personal Portfolio

A premium, minimalist personal portfolio website built by AbdulAziz Memon. This project utilizes a clean "Extreme UI" aesthetic with high-performance CSS and a focused white theme.

## Author
AbdulAziz Memon

## Tech Stack
- **HTML5**: Semantic and accessible document structure.
- **CSS3**: Custom design system featuring:
  - **Poppins Font**: Modern, high-readability typography.
  - **Glassmorphism**: Backdrop-filter effects on navigation.
  - **Advanced Layouts**: CSS Grid and Flexbox for responsiveness.
  - **Micro-interactions**: Smooth transitions and hover animations.

## Site Structure
- **index.html**: Professional hero section with high-contrast typography.
- **about.html**: Background on Xantrail Studio and development philosophy.
- **projects.html**: Interactive grid showcasing works like SatinScroll.js, BuildXpc, and ZanXOS.
- **contact.html**: Modern, styled inquiry form for collaborations.
- **style.css**: Centralized design system using CSS variables and Poppins integration.

## Design Philosophy
The site follows an "Editorial Tech" style:
- Maximum white space for clarity.
- Subtle shadows and rounded corners (24px) for a soft, premium feel.
- High-performance scrolling and hover states without using heavy libraries.

## Setup
1. Clone or download the directory.
2. Ensure `style.css` is in the same folder as the HTML files.
3. Open `index.html` in a browser to view.
