# Personal Portfolio

This is my personal portfolio website, created by myself (AbdulAziz Memon) using HTML and CSS. It is designed with a clean white theme and a minimalist aesthetic.

## Author
AbdulAziz Memon

## Pages
- index.html: The homepage featuring a hero section and introduction to my work.
- about.html: Information regarding my background as a developer and founder of Xantrail Studio.
- projects.html: A grid layout showcasing featured works like SatinScroll.js, BuildXpc, and ZanXOS.

- CSS3 (White Theme)
